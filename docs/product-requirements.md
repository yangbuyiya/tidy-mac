# 清洁王 产品需求与功能进度清单

更新时间：2026-06-07

## 1. 产品定位

### 1.1 产品名称

清洁王

### 1.2 一句话定位

一款面向 macOS 的本地安全文件清理工具，用于发现占空间、重复、长期未整理和指定后缀的文件，并通过系统废纸篓完成可恢复清理。

### 1.3 核心价值

- 本地扫描，不上传用户文件内容。
- 删除默认进入 macOS 废纸篓，降低误删风险。
- 扫描结果必须解释推荐原因。
- 高风险文件默认保护，不能批量删除。
- 支持用户配置扫描范围，避免固定目录不符合真实使用场景。

### 1.4 非目标

- 不做系统加速大师。
- 不做内存清理。
- 不做自动无确认删除。
- 不默认深扫系统敏感目录。
- 不承诺恢复已清空废纸篓的文件。
